# NeonVector Engine 🌟

レトロなベクターグラフィックスを現代的に再現したゲームエンジンライブラリ

## 📋 プロジェクト情報

- **バージョン**: 0.1.0
- **開発環境**: Visual Studio 2022, DirectX12
- **言語**: C++20

## 🎯 目標

1970-80年代のアーケードゲーム（Asteroids、Tempest、Battlezone）で使われた
ベクタースキャン技術をDirectX12で再現する。
